물리적 매체 없이 **전자기파(Electromagnetic Wave)** 를 이요해 에너지를 전달하는 방식.

# 전자기파의 원리 (The Physics)
**전자기파(EM Wave)** 는 **전기장(Electric Field)** 과 **자기장(Magnetic Field)** 이 서로 수직으로 진동하며 공간을 통해 **빛의 속도로** 전파해 나가는 에너지 형태.

#### 파동의 세 가지 핵심 속성
무선 통신은 데이터를 이 세 가지 속성에 실어 보냄.
1. 주파수 (Frequency, $f$): 1초에 파동이 몇 번 진동하는지를 나타냄. (단위: Hz, 헤르츠)
	- 역할: 통신 채널을 나눔. (예: Wi-Fi)는 2.4 GHz, 5GHz 주파수 대역을 사용함.)
2. 파장 (Wavelength, $\lambda$): 파동 한 번이 진행되는 길이. (주파수와 반비례: $c=f \cdot \lambda$)
3. 진폭 (Amplitude): 파동의 세기, 즉 **에너지의 크기**. (수신 강도, 거리에 영향을 줌.)

#### 전송 과정 (송신과 수신)
1. 송신 (Transmitting): 송신 안테나에 고주파 전류를 흘려보내면, 이 전류의 진동에 따라 주변에 전기장과 자기장이 생성되고, 이것이 스스로 복제되며 공간을 날아감.
2. 수신 (Receiving): 수신 안테나에 도달한 전자기파는 안테나 내부의 전자를 진동시켜 **다시 미세한 전류**를 발생시킴. 이 전류를 증폭하고 복조(Demodulation)하여 원본 데이터를 복원함.

# 데이터 변조 (Modulation)
전자기파는 그저 단순한 사인파(Sine Wave)입니다. 여기에 **데이터(정보, 0과 1)** 를 싣기 위해 파동의 속성을 변화시키는 기술을 **변조(Modulation)** 이라고 함.

| 방식              | 설명                                                    | 데이터 코딩 원리                    |
| --------------- | ----------------------------------------------------- | ---------------------------- |
| ASK (진폭 편이 변조)  | **파동의 진폭(크기)** 을 변경하여 데이터 표현.                         | 0: 진폭 낮게, 1: 진폭 높게           |
| FSK (주파수 편이 변조) | **파동의 주파수(진동 수)** 를 변경하여 데이터 표현.                      | 0: 낮은 주파수, 1: 높은 주파수         |
| PSK (위상 편이 변조)  | **파동의 시작 지점(위상)** 을 변경하여 데이터 표현.                      | 0: 0도, 1: 180도 위상            |
| QAM (복합 변조)     | **진폭** 과 **위상** 을 동시에 변조하여 한 번에 더 많은 비트(고속 데이터)를 전송함. | 현대 Wi-Fi. 4G/5G 등 고속 통신에 필수. |

> 결론: 우리가 쓰는 모든 무선 장치는 이 변조 기술을 이용해 0과 1의 디지털 데이터를 아날로그 전자기파에 실어 보내는 것.

# 무선 통신 규격 (Standards)
전자기파 위에서 데이터를 어떻게 주고받을지 정하는 **논리적 규칙(Protocol)**.

| 규격               | 역할             | 주파수 대역               | 특징                           |
| ---------------- | -------------- | -------------------- | ---------------------------- |
| Wi-Fi            | 근거리 무선 LAN     | 2.4 GHz, 5 GHz, 6GHz | 빠르고, 데이터 전송량이 많음 (TCP/IP 기반) |
| Bluetooth        | 근거리 저전력 개인 통신망 | 2.4 GHz              | 전력 소모가 매우 낮음 (IoT 장치 연결)     |
| Cellular (4G/5G) | 광범위 지역 이동 통신   | 수백 MHz ~ 수십 GHz      | 넓은 커버리지, 복잡한 기지국 시스템 필요      |

이러한 규격은 모두 **MAC 주소** 기반으로 인접 장치와 통신하며, 그 위에서 **TCP/IP** 프로토콜 스택을 사용하여 인터넷 통신을 수행함.
