- 진화 배경: 그래프에서 두 노드가 연결되어 있는지($O(1)$) 여부를 확인하고, 간선이 추가되거나 삭제될 때마다($O(1)$) 이 상태를 갱신해야 하는 실시간 네트워크/물리 시뮬레이션 문제.
- 핵심 원리:
	- Link-Cut Tree나 Euler Tour Tree 같은 복잡한 균형 트리 구조와 **Disjoint Set Union (서로소 집합)** 을 조합함.
	- 그래프의 구조가 변할 떄(간선 추가/삭제), 트리의 로직을 통해 전체 그래프의 연결 상태를 $O(\log n)$ 시간에 빠르게 갱신함.
- 결과: 실시간으로 간선이 추가되고 삭제되는 네트워크나 그래프 구조에서 두 지점 간의 연결 여부를 매우 빠르게 판단하고 갱신함.
