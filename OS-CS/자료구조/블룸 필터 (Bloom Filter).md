#### 블룸 필터 (Bloom Filter)
- 목표: 데이터베이스나 캐시 메모리에서 **"특정 데이터가 존재하는가?"** 라는 질문에 매우 빠르게 대답하기 위함임.
- 구조: 비트 배열과 여러 개의 해시 함수로 이루어져 있음.
- 원리: 데이터 하나를 저장할 때, 여러 개의 해시 함수를 거쳐 나온 여러 개의 인덱스에 해당하는 비트 배열의 값을 1로 설정함.
- 특징:
	- 'False Positive' (오탐) 허용: 블룸 필터가 "데이터가 있다"고 말했을 때 실제로는 없을 수 있음. (확률론적 오류)
	- 'False Negative' (미탐) 없음: 블룸 필터가 "데이터가 없다"고 말했을 때는 반드시 없는 것이 보장됨.
- 응용: 웹 브라우저가 접속하려는 URL이 '유해 사이트 목록'에 있는지 빠르게 확인하거나, 캐시에 데이터가 없는 경우(Cache Miss)를 빠르게 판단하여 디스크 접근 횟수르 줄이는 데 사용됨.
