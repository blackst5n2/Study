# 계산 그래프 (Computational Graph, DAG)
계산 그래프는 텐서(행렬)를 노드로 삼고, 이 텐서 간의 연산(덧셈, 곱셈 등)을 엣지로 연결한 구조임. 이는 현대 딥러닝 프레임워크(PyTorch, TensorFlow)가 작동하는 핵심 엔진임.

#### 계산 그래프의 정의 및 목표
- 정의: **방향성 비순환 그래프 (DAG, Directed Acyclic Graph)** 의 한 종류.
	- 노드 (Node): 데이터 (텐서) 또는 연산 (Operation, Function)
	- 엣지 (Edge): 데이터 흐름
- 목표: 딥러닝 모델의 복잡한 수식(수백만 개의 행렬 곱)을 자동으로 미분하여 **역전파 (Backpropagation)** 을 수행하는 엔진을 제공함.

#### 핵심 수학: 연쇄 법칙 (Chain Rule)
계산 그래프의 모든 연산은 **미분(Calculus)** 을 기반으로 함.
- 원리: 딥러닝 학습은 결국 **손실(Loss) 함수** 를 미분하여 가중치를 업데이트하는 과정. 복잡하게 연결된 수식을 미분할 때, 그래프를 거꾸로 따라가면서 각 노드(연산)에서 **연쇄 법칙(Chain Rule)** 을 적용해 미분 값을 전달함.
- 구현: 각 노드가 자신에게 들어온 입력값과 자신을 통해 나간 미분값을 기억하게 만들면, `backward()` 함수 한 번 호출로 전체 그래프의 미분 값이 자동으로 계산됨.
