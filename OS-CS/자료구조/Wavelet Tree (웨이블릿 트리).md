- 진화 배경: 문자열에서 **범위 쿼리(Range Query)** 를 수행하는 문제 (예: "이 문자열의 특정 범위내에 문자 'a'가 몇 번 나타나는가?").
- 핵심 원리:
	- **문자열(String)** 을 이진 트리 형태로 변환하여 저장함.
	- 각 노드는 문자의 집합을 이진 분할하여, 왼쪽 자식에는 '작은' 문자, 오른쪽 자식에는 '큰' 문자의 발생 횟수를 기록함.
- 결과: 압축된 공간을 사용하면서도, **순위 쿼리(Rank Query)** 나 **선택 쿼리(Select Query)** 같은 복잡한 문자열 통계 쿼리를 $O(\log \Sigma)$ (알파벳 크기)의 매우 빠른 시간 안에 처리할 수 있음.
- 응용: 압축된 대규모 텍스트 데이터베이스, 생물정보학.
