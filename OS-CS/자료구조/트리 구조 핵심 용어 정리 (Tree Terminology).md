트리 구조는 **가계도(Family Tree)** 나 조직도처럼 계층적인 관계를 모델링한 것.

#### 기본 구성 요소
| 용어                    | 정의                                           |
| --------------------- | -------------------------------------------- |
| 노드 (Node)             | 트리를 구성하는 기본 요소. 데이터와 다른 노드에 대한 연결 정보를 담고 있음. |
| 간선 (Edge)             | 노드와 노드를 연결하는 선. 계층적 관계를 나타냄.                 |
| 루트 노드 (Root Node)     | 트리에서 가장 꼭대기에 있는 노드. 부모가 없는 유일한 노드.           |
| 리프 노드 (Leaf Node)     | 자식 노드가 없는 노드. 트리의 가장 말단에 위치함.                |
| 내부 노드 (Internal Node) | 루트 노드이면서 리프 노드가 아닌 노드. 자식 노드를 가지고 있는 노드.     |

#### 관계 및 계층 용어
| 용어              | 정의                                            |
| --------------- | --------------------------------------------- |
| 부모 (Parent)     | 특정 노드 바로 위에 연결된 노드.                           |
| 자식 (Child)      | 특정 노드 바로 아래에 연결된 노드.                          |
| 형제 (Sibling)    | 같은 부모를 가지는 노드들.                               |
| 조상 (Ancestor)   | 특정 노드에서 루트까지 올라가는 경로상의 모든 노드 (부모, 할아버지 등 포함). |
| 자손 (Descendant) | 특정 노드 아래에 연결된 모든 노드 (자식, 손자 등 포함).            |
| 서브 트리 (Subtree) | 어떤 노드를 루트로 하여 그 노드와 자손들로 이루어진 부분 트리.          |

#### 크기 및 깊이 용어
| 용어          | 정의                                         |
| ----------- | ------------------------------------------ |
| 깊이 (Depth)  | 루트 노드에서 특정 노드까지의 간선 수. 루트의 깊이는 0임.         |
| 높이 (Height) | 특정 노드에서 가장 먼 리프 노드까지의 간선 수. 리프 노드의 높이는 0임. |
| 레벨 (Level)  | 같은 깊이를 가지는 노드들의 집합.                        |
| 차수 (Degree) | 노드가 가진 자식의 수.                              |

#### 균형 잡힌 트리 관련 용어 (Red-Black Tree 필수)
Red-Black Tree와 같은 균형 트리를 이해할 대 필요한 특별 용어.

| 용어                 | 정의                                                   | 역할                                                                    |
| ------------------ | ---------------------------------------------------- | --------------------------------------------------------------------- |
| 삼촌 (Uncle)         | 부모 노드의 형제 노드. (나의 할아버지의 자식 중 나의 부모가 아닌 노드)           | Red-Black Tree에서 삽입 후 균형을 잡을 때 색깔 변경 (Recoloring) 여부를 결정하는 중요한 기준이 됨. |
| 할아버지 (Grandparent) | 부모 노드의 부모 노드. (나의 부모 노드 바로 위의 노드)                    | Red-Black Tree에서 회전(Rotation)이 일어날 때 기준이 되는 노드임.                      |
| NIL 노드             | 실제 데이터는 없지만, 트리의 끝을 나타내는 가상의 Black 리프 노프 (RBT 규칙 3). | 모든 노드가 정확히 2개의 자식을 가진 것처럼 취급하여 RBT 규칙을 단순화함.                          |
